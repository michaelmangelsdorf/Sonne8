This page describes the following instruction:

  • Group: GIRO
  • Mnemonic: g0
  • Opcode: 72d, 48h
  • Encoding: 01_00_1_000b (GIRO_REG_DIRECTION_INDEX)
  • Pseudo-code: L0 = g; pc++

Puts G into local memory at offset 0xF8.
